> Arvis workflow to beautify JSON/XML in clipboard


## Usage

### Copy JSON/XML snippet to clipboard

- <kbd>cmd</kbd> + <kbd>C</kbd>


### Beautify

- Type `tidy`, <kbd>Space</kbd>, select JSON or XML
- <kbd>Enter</kbd>
- <kbd>Cmd</kbd> + <kbd>V</kbd> to your editor
